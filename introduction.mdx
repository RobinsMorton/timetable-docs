## Getting TimeTable Visio

The first step to world-class TimeTables is downloading and installing required software.

<CardGroup cols={2}>
  <Card
    title=" Get Microsoft Visio"
    icon="Microsoft"
    href="https://www.microsoft.com/en-us/microsoft-365/visio/visio-plan-2"
  >
    Install Microsoft's Diagraming Tool. Online Plan 2 recommended.
  </Card>
  <Card
    title="Install TimeTable"
    icon="download"
    href="https://firebasestorage.googleapis.com/v0/b/timetable-downlaods.appspot.com/o/InstallTimeTableVisioAlpha.exe?alt=media&token=87302841-1cf5-47ad-9aa9-a9cc4b534ed0"
  >
    TimeTable is a plug-in to Microsoft Visio. After this is installed, you'll need a license.
  </Card>
</CardGroup>
